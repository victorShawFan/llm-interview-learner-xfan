> 一个系统化的大模型算法工程师面试复习资料库

## 📚 项目简介

本项目旨在帮助准备大模型（LLM）算法工程师面试的同学系统化复习相关知识，涵盖：
- 深度学习与NLP基础知识
- 预训练模型（BERT、GPT、T5等）
- Code Agent开发
- 算法题（LeetCode高频题）
- 面试真题与技巧

## 🎯 学习路径

### 初级阶段（1-2周）
1. 掌握Transformer架构和Attention机制
2. 理解预训练-微调范式
3. 刷50道LeetCode中等难度题

### 中级阶段（3-4周）
1. 深入理解BERT、GPT的区别和应用
2. 学习Agent开发基础
3. 掌握动态规划、图算法

### 高级阶段（5-6周）
1. 研究最新大模型技术（MoE、长上下文等）
2. 实践Code Agent项目
3. 模拟面试和总结
